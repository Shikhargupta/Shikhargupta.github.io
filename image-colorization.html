Image Colorization

Project Description

This project is a web application that uses a deep neural network to colorize images. The user can upload a grayscale or a black and white image and get the colorized version as an output. The model is hosted on Amazon Web Services (AWS) and deployed using Heroku. Link to the Heroku app.

Model Architecture

The deep learning model used for inference is a convolutional neural network (CNN) trained over a million color images. It was developed and trained by Richard Zhang. Below is the overall architecture of the model

More details about the model can be found on the project page.

System Architecture

  • The system architecture consists mainly of 2 components - AWS and Heroku
  • AWS hosts the deep learning model. Onnx runtime is used for inference.
  • Heroku hosts a REACT web application which takes in the user input and makes a POST request to AWS to get the colorized image.
